In this visualization task, I would analysis three topics: the first one is to show the distribution of total cases of the COVID-19 over the wolrd; the second one is that showing the new cases condition of each country; The last one is to show the total deaths cases of each country.
In this first part, it would use two dataset to finish the visualzaition task, the first one is a vector data which is used to draw the world map, the data source link is:

https://www.trafforddatalab.io/interactive_graphics_companion/data/countries.geojson

and the another dataset is the COVID-19 data which is used to analysis the distribution of the COVID-19 over the world, its data source link is:

https://github.com/owid/covid-19-data/blob/master/public/data/owid-covid-data.csv

From the color map above, the color of each region in the world map is different, some region's color is deep, and some region's color is light, that's because I use the depth of the color to represent the population of the total cases of the COVID-19 in each country. From the color world map, it obviously know the total cases population of the region with deep color are more serious than other region, especially, the Brazil, USA, India and Russia, and so on.

In this part, it would show the new cases of the COVID-19 of each country. The different country use the different color to mark, and it use the new cases value to represent the size of the circle, the larger of the new cases, the larger the circle's radius. Additionally, it would filter all the new cases according to the region. Meanwhile, it could slide the slide bar to see different new cases of different country.
The bar chart above is used to show total deaths cases of each country. The longer the bar chart, the more deaths cases. Of course, we could filter what we want to see. For example, it could choose one region, the chart above would just show all the bar charts of the countries belonging to the region. Meanwhile, it also would slide the range bar to see the bar charts among different numeric values.
